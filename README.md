# rxscrape

A tool leveraging Playwright for efficient web scraping.

## Getting Started

### Prerequisites

- Node.js (v16+)
- npm

### Installation

```bash
git clone https://github.com/walsh-kang/rxscrape.git
cd rxscrape
npm install
```
